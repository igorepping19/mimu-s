# 🐾 Mimu's Pet Shop — Landing Page

Landing page desenvolvida para apresentar os serviços e produtos da **Mimu's Pet Shop**, uma loja voltada ao cuidado e bem-estar de pets localizada em **Chapecó – SC**.

O projeto foi criado com foco em **layout moderno, responsividade e navegação simples**, servindo também como **projeto de prática em desenvolvimento front-end**.

---

# 🚀 Funcionalidades

- Menu fixo com navegação entre seções
- Botão de contato direto com **WhatsApp**
- Sessão de apresentação da loja
- Cards de produtos
- Galeria de serviços
- Sessão de avaliações
- Call To Action para contato
- Layout responsivo para diferentes telas

---

# 🧩 Estrutura da Página

O site é organizado nas seguintes seções:

### Hero
Apresentação inicial da loja com chamada para visualizar produtos.

### Benefícios
Destaque de vantagens da loja:

- Compra segura  
- Atendimento rápido  

### Nossa Loja
Apresentação da loja física com endereço e informações de contato.

### Produtos
Cards com categorias principais de produtos para pets.

### Serviços
Galeria com imagens de serviços realizados.

### Avaliações
Seção com feedback visual de clientes.

### CTA (Call To Action)
Convite para contato via **WhatsApp**.

### Footer
Informações finais da loja e localização.

# 🛠 Tecnologias Utilizadas

- **HTML5**
- **CSS3**

# 📱 Responsividade

O site foi desenvolvido para funcionar em diferentes tamanhos de tela:

- Desktop
- Tablet
- Smartphone

Utilizando **Media Queries** para adaptação do layout.

Este projeto foi criado com o objetivo de:
